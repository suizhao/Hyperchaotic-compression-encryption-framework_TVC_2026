# High fidelity image compression-cryptosystem
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains the implementation code for the paper:
**High fidelity image compression-cryptosystem via 2D coupled complex-argument map and square scrambling**
(Manuscript ID: e718b25a-0c2d-4d82-afdc-92b44d8985fa)

## Overview
This project implements a high-fidelity image compression-cryptosystem based on 2D coupled complex-argument map and square scrambling. The code includes complete encryption framework, decryption framework, and test programs, which can reproduce the experimental results in the paper.

## Environment Requirements
- MATLAB R2024b or later (compatible with most versions after R2018a)

## File Structure
├── encryption_framework.m  # Encryption Framework
├── decryption_framework.m  # Decryption Framework
├── energy_ac_coefficients_framwork.m # Outline Attack Framework
├── main.m                  # Test Process
├── *.mat, *.png and *.tiff  # Test Data
└── README.md                # Document
